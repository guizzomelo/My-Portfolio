Rede social Opinion Share

O objetivo deste projeto é criar uma espécie de rede social onde as pessoas logadas podem compartilhar as suas opiniões em um chat global, usando a API do YouTube. Você pode pesquisar por um vídeo e compartilhar no chat global para receber comentários de outras pessoas. A aplicação não está totalmente concluída agora, só falta o backend e corrigir alguns bugs, porém consegui progredir muito usando as tecnologias HTML, CSS/tailwind, Javascript/ReactJS, Firebase.
